# 🏨 Hotel Management System  
A fully responsive and clean Hotel Management System website built using *HTML, CSS, and JavaScript*.  
This project showcases different hotel rooms, services, booking pages, login form, and a smooth user interface.

---

## 🚀 Features

- ✔ Beautiful and responsive UI  
- ✔ Homepage with navigation bar  
- ✔ Room categories with images  
- ✔ Services page  
- ✔ Reservation/booking page  
- ✔ Customer login form  
- ✔ Image slider / gallery  
- ✔ Smooth animations and transitions  
- ✔ Welcome alert on first visit  

---

## 🛠 Technologies Used  

- *HTML5* – Web page structure  
- *CSS3* – Styling & responsive design  
- *JavaScript* – Interactivity and dynamic behavior
